# Employee Payroll System

## 💼 Employee Payroll Management System

A full-stack web application designed to manage employee records, generate payroll, and calculate salaries efficiently. The system provides an interactive dashboard with employee management features and automated payroll calculations.

---

## 🚀 Live Demo

🌐 https://employee-payroll-managementsystem.netlify.app/

---

## 📌 Features

- ➕ Add new employees
- 📋 View employee records
- ❌ Delete employees
- 💰 Generate payroll
- 🧮 Automatic net salary calculation
- 📊 Real-time payroll summary
- 🌗 Light/Dark mode toggle
- 🛡️ Backend validations
- 🔄 REST API integration

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Deployment
- Frontend: Netlify
- Backend: Render
